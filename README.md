#Social Media Analytics for Strategic Branding

This project presents an end-to-end Social Media Intelligence Solution developed for PepsiCo as part of a data analytics learning journey. The solution consolidates, cleans, and analyzes data from multiple social media platforms (Instagram, Twitter, Facebook, YouTube) to drive actionable insights and optimize marketing performance.

Key Features:

Comprehensive data preprocessing and cleaning (duplicates removal, format standardization)

Calculation of engagement metrics and identification of top-performing posts and hashtags

Platform-wise analysis of follower growth, engagement, and advertising ROI

Impactful data visualizations (pivot tables, charts, and trend analysis)

Campaign effectiveness measurement, including uplift, ROI, and retention

Actionable insights and recommendations for optimizing social media content and strategy

Tech Stack:
Excel, Data Analytics, Pivot Tables, Power Qqerry Editor
